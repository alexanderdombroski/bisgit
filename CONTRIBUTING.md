# Thanks for trying out my git client!

## Things you need installed

- [Node JS](https://nodejs.org/en)
- [Git cli](https://git-scm.com/)
- [Github cli](https://cli.github.com/)

## Running the app

Install dependencies with `npm i`

### Dev with hot reload

1. `npm run build` in one terminal
2. `npm run dev` in another terminal

### Production build

1. `npm run package`
2. `npm run start`

## Debug with VS Code

- ensure you have extensions installed in [extensions.json](./.vscode/extensions.json)
- Press `debug tui` in the [Run and Debug menu](https://code.visualstudio.com/docs/debugtest/debugging)
